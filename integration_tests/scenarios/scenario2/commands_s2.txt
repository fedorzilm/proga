# Чтобы этот тест был осмысленным, в вашем initial_data_s2.txt должно быть несколько записей
# с ОДИНАКОВОЙ датой, например, "01.01.2024" для первых 2-5 записей.
# Если все даты уникальны, этот SELECT вернет 0 или 1 запись.
